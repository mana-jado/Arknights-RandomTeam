# 明日方舟干员随机选择器

一个基于网页的明日方舟干员随机选择工具，可以读取您的干员数据JSON文件，随机选择12名干员并生成PRTS.ME兼容的配置。

## 功能特性

- 📁 **文件上传**：支持拖放或点击选择JSON文件
- 🎲 **智能随机**：提供两种抽取模式（简单随机/等级权重）
- ⚙️ **过滤选项**：可忽略精英0级且等级为1的干员
- 🖼️ **图片显示**：显示抽取干员的图片（需预先准备图片资源）
- 📊 **实时统计**：显示抽取结果中的6星干员数量
- 💾 **导出功能**：生成并下载符合PRTS.ME标准的JSON文件
- 📱 **响应式设计**：适配各种屏幕尺寸
- 🎨 **美观界面**：现代化UI设计，深色主题
- AI，AI，还是tmd AI
